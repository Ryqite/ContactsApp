# ContactsApp
Technologies I used: Android Sdk, Jetpack Compose, Clean Architecture, Coil
--------------------------------------------------------------------------------------
This is my Contacts app. Here I tried to work with Android SDK Components
--------------------------------------------------------------------------------------
